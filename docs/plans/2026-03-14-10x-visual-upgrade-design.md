# 10x Visual Upgrade — "Matrix Boot Sequence"

**Date:** 2026-03-14
**Status:** Approved

## Overview

Upgrade Agent Wojak from functional to unforgettable with premium animations using framer-motion. Zero cost. Five areas: boot-up intro, page transitions, scroll-triggered reveals, interactive hovers, micro-interactions.

## 1. Boot-Up Intro Animation
Full-screen terminal boot sequence on first visit (~3s). Typed lines: INITIALIZING, CONNECTING TO SOLANA, LOADING FEELINGS, STATUS ONLINE, ENTERING THE MATRIX. Dissolves into site. Plays once per session (sessionStorage).

## 2. Page Transitions
Fade + slide between pages using framer-motion AnimatePresence. Current page fades out left, new page fades in from right. Matrix rain persists.

## 3. Scroll-Triggered Reveals
Every section animates in on scroll — fade up + scale. Staggered children. Gallery cascade. Stats count up from 0.

## 4. Interactive Hover Effects
Gallery: 3D tilt + glow. Navbar: glitch on hover + sliding underline. Buttons: scale + glow pulse. Cards: animated border trace. Hero image: floating bob animation.

## 5. Micro-interactions
Chat messages slide in. Meme generation Matrix decode effect. Smooth scroll. Typing indicator with scrambled characters.

## Tech
framer-motion (free, ~40KB). CSS for simple stuff. Zero API costs.
